## How to use a template

1. Duplicate the template file.
2. Move the copy into the matching folder.
3. Rename it to a slug: lowercase, hyphen-separated (e.g. `zettelkasten-method.md`).
4. Update the frontmatter (`title`, `tags`, `created`).
5. Fill in each section, deleting any you don't need.
6. Link the new note from the folder's `index.md`.
